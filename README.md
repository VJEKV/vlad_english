# VladEnglish — Приложение для изучения английского языка

## О проекте

**VladEnglish** — десктопное приложение для Windows для изучения английского языка школьниками 7–14 лет (2–8 класс) по линейке учебников Spotlight. Приложение обучает чтению (phonics), лексике, грамматике, аудированию и письму. Все объяснения — на русском языке с озвучкой.

На текущий момент полностью реализован контент для **2 и 3 класса** (Spotlight 2 и Spotlight 3).

## Содержание приложения

### Phonics — 47 уроков обучения чтению

Каждый урок состоит из 4 интерактивных этапов:
1. **Изучаем** — большая буква/сочетание + звук (медленная озвучка) + слово-пример с эмодзи
2. **Узнай звук** — слышишь звук → выбираешь букву из 3 вариантов
3. **Подбери слово** — видишь картинку и перевод → выбираешь английское слово
4. **Собери слово** — слышишь слово → собираешь из букв-плиток

| Раздел | Уроков | Что изучается |
|--------|--------|---------------|
| Алфавит | 5 | Буквы A–Z, звуки, слова-примеры |
| Короткие гласные | 5 | Short A, E, I, O, U — по 10 CVC-слов |
| Сочетания согласных | 9 | bl/br, cl/cr, dr/fl, fr/gl, gr/pl, pr/sl, sm/sn, sp/st, sw/tr |
| Долгие гласные (Magic E) | 5 | a_e, i_e, o_e, u_e, e_e — по 10 слов |
| Диграфы | 7 | sh, ch, th, wh, ph, ck, ng — по 10 слов |
| Сочетания гласных | 6 | ee, ea, ai/ay, oa, oo + повторение |
| R-контролируемые | 5 | ar, er, ir, or, ur — по 10 слов |
| Sight Words | 5 | 50 высокочастотных слов |

Под списком уроков — русские объяснения правил чтения с озвучкой:
- **TH** — «Положи язык между зубами и подуй...»
- **SH** — «Как русское Ш, но мягче...»
- **CH** — «Как русское Ч — легко!»
- **Short A** — «Между русскими Э и А...»
- **Magic E** — «Немая E в конце слова делает гласную длинной...»

**Каждый модуль Spotlight — полноценный урок из 6 этапов:**
1. Изучи слова (карточки с озвучкой)
2. Проверь себя (выбери перевод)
3. Напиши слово (услышь → напечатай)
4. Прочитай предложения (с озвучкой)
5. Грамматика модуля (правила + фразы)
6. Итоговый тест

### Грамматика — 5 тем с русскими объяснениями

Каждая тема содержит:
- Объяснение на русском с озвучкой
- Аналогия с русским языком («В русском это как...»)
- Формула (am/is/are + verb-ing)
- Примеры с подсветкой ключевых слов
- Частые ошибки с разбором
- Совет для запоминания

| Тема | Класс | Описание |
|------|-------|----------|
| To Be (am/is/are) | 2–3 | «В русском пропускаем, в английском — нельзя» |
| Can / Can't | 2–3 | «Умею/не умею — одинаковый для всех» |
| Present Simple | 3–4 | «Что делаю обычно, каждый день» |
| Present Continuous | 3–4 | «Что делаю прямо сейчас» |
| Past Simple | 4–5 | «Что уже произошло — вчера, на прошлой неделе» |

### Чтение — 16 текстов

Тексты для чтения с интерактивными функциями:
- Нажми на любое слово — услышишь произношение
- Нажми на предложение — озвучка целиком
- Кнопка «Послушать весь текст»
- Скрытый перевод — показывается по клику

Примеры текстов: My Cat, At School, My Family, In the Park, My Dog, Breakfast, My Room, At the Beach, My Best Friend, At the Zoo, My Birthday Party, Our School, A Rainy Day, Shopping, My Pet Hamster.

### Аудирование

- 70+ слов из словаря Spotlight 2–3
- 10 случайных вопросов за сессию
- Слышишь слово → выбираешь перевод из 3 вариантов

### Письмо — 30 предложений

Переведи с русского на английский:
- 10 лёгких (2 класс): «У меня есть кот» → I have a cat
- 10 средних (2–3 класс): «Кот под столом» → The cat is under the table
- 10 сложных (3 класс): «Она обычно встаёт в семь» → She usually gets up at seven

Есть подсказка (шаблон с пропусками) и проверка.

### Игры — 3 вида

| Игра | Описание |
|------|----------|
| **Memory Cards** | Найди пары слово ↔ перевод (12 карточек) |
| **Grammar Battle** | 60 секунд — выбирай is/are/am, can, has/have (61 вопрос) |
| **Spelling Bee** | Послушай слово — напиши его (70+ слов) |

## Система прогресса

- **Звёзды** (1–3) за каждый урок в зависимости от процента правильных ответов
- **Очки** — 10 за правильный ответ, бонус 50 за 3 звезды
- **Streak** — дней подряд
- Прогресс сохраняется в JSON-файл на диске

---

## Адаптация под возраст

При первом запуске ученик выбирает класс (2–8). В зависимости от класса:
- Показываются только доступные разделы (Phonics — для 2–3 кл, Grammar — для 3+ кл)
- Sidebar фильтрует навигацию
- Spotlight показывает контент нужного учебника

---

## Озвучка

Вся озвучка через Web Speech API (работает офлайн в Windows):
- **Буквы**: rate 0.4, pitch 1.1 — очень медленно и чётко
- **Слова**: rate 0.55 — медленно
- **Предложения**: rate 0.7 — умеренно
- **Русские объяснения**: rate 0.9, lang ru-RU
- **Spell mode**: побуквенно c-a-t → cat
